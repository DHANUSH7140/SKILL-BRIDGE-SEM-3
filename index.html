About SkillBridge

SkillBridge solves the fragmentation problem between volunteers and organizations. Instead of scattered WhatsApp groups and forms, organizations can publish structured opportunities, and volunteers can browse, filter, and apply from a single mobile app.

Tech stack

The project is built as a full-stack solution:

  • Frontend: Flutter (Dart), Material 3, responsive layouts
  • Backend-as-a-Service: Supabase (Auth, REST, Realtime)
  • Database: Postgres with opportunities table
  • Platform: Android APK build (college demo ready)
